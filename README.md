# Dicionario_hamu


Um dicionário interativo desenvolvido para gerenciar, expandir e consultar o vocabulário do **Hamu**, minha língua fictícia (conlang). Este projeto nasceu da necessidade de centralizar o léxico e evitar que eu me perca em meio a tantas variações e formas de escrita.

## Funcionalidades

- **Busca Bilíngue:** Pesquise termos tanto em Hamu quanto em Português para obter a tradução instantânea.
- **Gerenciamento de Léxico:** Adicione novas palavras facilmente para expandir o vocabulário da língua.
- **Centralização:** Um único ponto de verdade para manter a consistência ortográfica e semântica do idioma.
